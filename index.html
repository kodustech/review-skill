Review Skill

Code review com IA direto na sua IDE. Integra o Kodus CLI em +20 ferramentas de desenvolvimento.

Instalacao
curl -fsSL https://review-skill.com/install | bash

Comandos

kodus review

Executa code review nas suas alteracoes. Analisa o codigo, identifica problemas e aplica correcoes automaticamente.

--prompt-only --staged --commit <sha> --branch <name> --fast
kodus pr suggestions

Analisa sugestoes de PR e aplica melhorias de baixo risco. Pula mudancas irrelevantes ou que mudam o escopo.

--pr-url <url> --pr-number <n> --repo-id <id>
